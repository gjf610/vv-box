# VV-Box - 一个 Vue UI 组件


## 介绍

这是我在学习 Vue 过程中做的一个 UI 框架，希望对你有用。

## 开始使用

1. 安装 VV-box
  ```
  npm i --save VV-box
  ```
3. 引入 gulu
  ```
  import {Button, ButtonGroup, Icon} from 'VV-box'
  export default {
    name: 'app',
    components: {
      'vv-button': Button,
      'vv-icon': Icon
    }
  }
  ```
